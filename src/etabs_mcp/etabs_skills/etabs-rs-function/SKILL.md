# ETABS Response Spectrum Functions

## Tables

| Table | Use |
|---|---|
| `Functions - Response Spectrum - ASCE7-05` | ASCE 7-05 parametric RS (IBC 2006/2009/2012, BNBC 2020) |
| `Functions - Response Spectrum - User Defined` | Custom period-value pairs |

---

## ASCE 7-05 RS Function

### Fields (ETABS 23.2.0 — verified)

```
Name, Period, Value, SsAndS1From, Ss, S1, TL, SiteClass, Fa, Fv, SDS, SD1, DampRatio, GUID
```

### Row structure (critical)

Each function has **one parameter row** (Period=0) + N auto-generated data rows:

| Row | Period | Value | Other fields |
|---|---|---|---|
| Row 0 (parameter) | 0 | 0.4·SDS | All params populated |
| Row 1–N (curve) | T | Sa(T) | All null — auto-generated by ETABS |

**To create a function:** write ONLY the parameter row (Period=0) with all parameters. ETABS auto-generates the spectrum curve. Do not pre-compute period-value pairs yourself.

### Key field values — VERIFIED from live ETABS 23.2.0 model

| Field | Value | Note |
|---|---|---|
| `SsAndS1From` | `"User Defined"` | Space required — "UserDefined" is rejected |
| `SiteClass` | `"F"` | Always "F" — ETABS overwrites Fa/Fv for A–E |
| `Period` | `"0"` | Must be string "0" for the parameter row |
| `Value` | `str(0.4 * SDS)` | Sa at T=0 = 0.4·SDS per ASCE 7-05 §11.4 |
| `DampRatio` | e.g. `"0.05"` | Typically 5% |

### Verified example — existing function "ASCE 7-05/bnbc" from live model

```
Ss=0.5, S1=0.2, TL=4, SiteClass=F, Fa=1.4, Fv=1.75
SDS = 2/3·1.4·0.5 = 0.4667
SD1 = 2/3·1.75·0.2 = 0.2333
T0  = 0.2·SD1/SDS = 0.1 s
Ts  = SD1/SDS = 0.5 s
```

Auto-generated curve: T=0→0.187, T=0.1→0.467, T=0.5→0.467, T=0.8→0.292, T=1→0.233, T=2→0.117, T=4→0.058, T=10→0.009

### Safe write pattern (Python / MCP)

```python
TABLE = "Functions - Response Spectrum - ASCE7-05"
model.SetModelIsLocked(False)

# Seismic parameters
Ss=0.5; S1=0.2; TL=4; Fa=1.35; Fv=2.7; damp=0.05
SDS = round(2/3*Fa*Ss, 6)
SD1 = round(2/3*Fv*S1, 6)
func_name = "RS-ASCE705"

# Read full existing table — use ETABS's own field list
t = model.DatabaseTables.GetTableForDisplayArray(TABLE, [], "All", 0, [], 0, [])
fields = [f for f in list(t[2]) if f is not None]
n = t[3]; nf = len(fields); flat = list(t[4])
rows = [{fields[j]: flat[i*nf+j] for j in range(nf)} for i in range(n)]

# Drop only rows belonging to the function being (re)created
rows = [r for r in rows if r.get("Name") != func_name]

# Build parameter row (Period=0) — ETABS auto-generates full spectrum curve
new_row = {f: "" for f in fields}
new_row.update({
    "Name": func_name,
    "Period": "0",
    "Value": str(round(0.4 * SDS, 6)),
    "SsAndS1From": "User Defined",
    "Ss": str(Ss), "S1": str(S1), "TL": str(TL),
    "SiteClass": "F",
    "Fa": str(Fa), "Fv": str(Fv),
    "SDS": str(SDS), "SD1": str(SD1),
    "DampRatio": str(damp),
})
rows.append(new_row)

# Write back using ETABS's own field list
flat_out = tuple(r.get(f, "") or "" for r in rows for f in fields)
ret_s = model.DatabaseTables.SetTableForEditingArray(TABLE, 0, fields, len(rows), flat_out)
ret_a = model.DatabaseTables.ApplyEditedTables(True)
result = {"staged": ret_s[0], "errors": ret_a[0], "warnings": ret_a[1]}
# Expected: staged=1, errors=0, warnings=0
```

**MCP-tested**: staged=1, errors=0, warnings=0. ETABS returned 25 curve points after apply.

## User-Defined RS Function

### Fields

```
Name, Period, Value, DampRatio, GUID
```

### Write pattern

Provide all period-value pairs explicitly (one row each).

```python
TABLE = "Functions - Response Spectrum - User Defined"
t = model.DatabaseTables.GetTableForDisplayArray(TABLE, [], "All", 0, [], 0, [])
fields = [f for f in list(t[2]) if f is not None]
n = t[3]; nf = len(fields); flat = list(t[4])
rows = [{fields[j]: flat[i*nf+j] for j in range(nf)} for i in range(n)]
rows = [r for r in rows if r.get("Name") != "MY_RS"]

curve = [(0, 1.0), (0.5, 1.0), (1.0, 0.8), (2.0, 0.4), (4.0, 0.2)]
for T, Sa in curve:
    r = {f: "" for f in fields}
    r.update({"Name": "MY_RS", "Period": str(T), "Value": str(Sa), "DampRatio": "0.05"})
    rows.append(r)

flat_out = tuple(r.get(f, "") or "" for r in rows for f in fields)
ret_s = model.DatabaseTables.SetTableForEditingArray(TABLE, 0, fields, len(rows), flat_out)
ret_a = model.DatabaseTables.ApplyEditedTables(True)
```

## ASCE 7-05 Spectrum Equations (§11.4)

```
SDS = 2/3 · Fa · Ss
SD1 = 2/3 · Fv · S1
T0  = 0.2 · SD1/SDS
Ts  = SD1/SDS

Sa(T) = SDS·(0.4 + 0.6·T/T0)   for  0 ≤ T ≤ T0
Sa(T) = SDS                      for  T0 ≤ T ≤ Ts
Sa(T) = SD1/T                    for  Ts ≤ T ≤ TL
Sa(T) = SD1·TL/T²               for  T > TL
```

## C# Plugin — RsCreateFunction()

Located in `Form1.cs`, tab "RS Function". Key points:
- Reads full table with ETABS's own field list via `GetTableForDisplayArray`
- Drops only rows matching the target function name
- Appends single parameter row (Period=0) with all params
- Calls `SetTableForEditingArray` + `ApplyEditedTables`
- SiteClass hardcoded "F", SsAndS1From = "User Defined"
- Live-computed SDS/SD1/T0/Ts display updates as user types

---

## Notes

- **Never** set SiteClass to A–E — ETABS overwrites Fa/Fv with its built-in table for those classes.
- After `ApplyEditedTables`, read back to verify: check that curve rows appeared (Period > 0 rows with null other fields).
- Verified on ETABS 23.2.0, model `D:\Atibazar\etabs2\Ati-N-15.$et`.
